# Introduction
This is the github repository of school project called CartoonHouse. It is a fictive restaurant that sells different imaginary foods from cartoon, movies and games. It has both the frontend and backend to the website.

# Installation
1. Clone the repository
2. Open the project in your IDE.
3. Create an empty MariaDB database.
4. Configure the database connection in db/secret-diary-db.sql
5. Use npm install to install the required dependencies.
6. Run the application using npm run dev.
7. Open the browser and go to http://localhost:5501.
8. The website is now running.

# Instructions
1. Run the application
2. Click start to be redirected to the home page.
3. On the home page you can register a new account or log in if you already have an account.
4. Home page displays some info about the restaurant as well as showing some discounted foods for registered users.
5. In the menu page the user can browse the menu as well as add food items into the shopping cart.
6. Inside the shopping cart is the button to go to the checkout.
